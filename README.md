## Welcome to AccurateClock Chrome Extension!   
The AccurateClock Chrome extension brings the beloved website to your browser!   
This README covers installation, usage, tech stack, licensing, and has a quick preview.    

## Installation
To install, download the latest release of the extension, and then navigate to ```chrome://extensions```   
Then, enable Developer Mode if not already blocked (workplaces and school may block Dev Mode from being enabled)   
Then, cick on 'Load Unpacked' and select the release folder.    
QuickTip! Because GitHub releases can be wonky, you might need to navigate to a folder in a folder, hence just keep on going into folders until you see the manifest.json and icons/.        
Once that is done, this is optional, but pin the extension.     

## Usage
Usage is fairly simple, just click on the extension and boom! Same UI as the website, well cause it is a website.     
You'll find out in tech stack.    

## Tech Stack

* CSS  
* JS
